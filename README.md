# College Event Manager

A comprehensive web portal for managing college events where students can register for various events organized by the college.

## Features

- User Authentication & Security (JWT, Google reCAPTCHA, IP rate limiting)
- Event Management (Solo & Team event registration)
- User Profiles & Participation Tracking
- Team Invitation System

## Technology Stack

- **Frontend**: React.js
- **Backend**: Node.js
- **Authentication**: JWT (JSON Web Tokens)
- **Security**: Google reCAPTCHA, IP Rate Limiting
